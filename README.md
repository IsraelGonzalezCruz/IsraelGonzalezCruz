### 🔬 [Aceros FIS - Calculadora Inteligente de Aceros al Carbono](https://github.com/IsraelGonzalezCruz/aceros-fis-app)
**Python | Streamlit | Análisis de Datos | Machine Learning**

Aplicación web interactiva para selección de aceros al carbono basada en propiedades mecánicas y tratamientos térmicos.

**Características:**
- 📊 Análisis estadístico de 190 registros (7 grados SAE)
- 🔍 Sistema inteligente de recomendación con 3 modos de usuario
- 📈 Visualizaciones interactivas (Plotly, Seaborn)
- 🧪 ANOVA bifactorial para validación científica
- 💾 Exportación de resultados para análisis posterior
- 🎯 Aplicaciones prácticas: domésticas, startups/makers, mecatrónica

**Impacto:** Herramienta que va más allá de requisitos académicos, diseñada para uso profesional real.
